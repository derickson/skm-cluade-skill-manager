# project claude code skill manager

this will be an applicaiton coded in python which will manage my claude code skills
This local project will be where I keep all of my claude code skills commands and other setups checked into the ./claude/skills .claude/commands etc folders
There will also be a small python program meant to be used from the command line which I can add to the path of my dev laptops
There will be a claude code skill for claude-skill-manager which will act as a text based interface for navigating and searching my library of skills.  

there will be a command from outside of claude code to install the skill manager skill inside a local claude code project
from inside claude code that skill will prompt me for skills to install, copying them from my checkout of this project (currently in /Users/dave/dev/claude-skill-manager) to the local project in question
from inside claude code there will be an option to navigate the existing skills and commands and save a copy of them to my claude-skill-manager setup (with warnings for overwriting anythign already there)
